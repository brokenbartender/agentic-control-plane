# Agentic Control Plane

Control-plane repo for Codex CLI skills and multi-node orchestration.

## Scope
- Skills for building/testing Enterprise
- Repo-scoped exec via codex-bridge
- UI automation via control stack

## Core Skills (must-have)
- build_enterprise
- test_enterprise
- start_enterprise_dev
- run_e2e
- ui_automation
- repo_exec

## Node Roles
- desktop: coordinator (always on)
- work: secondary node

## A2A Notes
Configured in AgenticConsole `.env` on each node. See `docs/a2a.md`.
